How It Works

From zero to posted comment in 3 steps

No complicated setup. Just install, configure your AI, and start commenting.

1

Install & Configure

Load the extension in Chrome, then set up your AI provider — OpenAI API key or a local Ollama endpoint.

2

Open a LinkedIn Post

Click the comment box under any post. The tone selector and generate button appear automatically.

3

Pick Tone & Generate

Select a tone from the pill bar, click 💬 Generate with Commently, and your comment is ready.

4

Review & Post

The comment is inserted into the box. Edit if needed, then hit Post — done.

FAQ

Frequently asked questions

Is Commently free?
Yes — Commently is completely free for personal and non-commercial use. You only pay for the AI service you connect (e.g. OpenAI API usage). Running a local LLM has no cost at all.
Does it store my LinkedIn data?
No. Commently only reads the post text from the page you're currently viewing and sends it to your chosen AI provider (OpenAI or your local LLM). Nothing is stored on any server. In local LLM mode, nothing leaves your machine.
Why isn't the generate button appearing?
Check that Commently is enabled (click the toolbar icon — status should show ✅ Ready). Refresh the LinkedIn page. Make sure you clicked inside the comment text box, not just near it. If it still doesn't appear, check the browser console (F12) for errors.
Can I use it without an OpenAI key?
Yes! Set it to Local LLM mode and connect Ollama or LM Studio. No API key needed. See the local LLM guide for setup instructions.
The comment has preamble like "Here's a comment:"
Commently automatically strips preamble lines from the AI's response. If it still appears, try a different model — instruction-tuned models (e.g. mistral) are much better at following the "output only the comment" instruction.
Can I use Commently commercially?
Commercial use (monetising, white-labelling, selling the extension) requires written permission from AgentCoreX. Contact agentcorex26@gmail.com to discuss a commercial license.
How do I contribute?
Fork the repo on GitHub, make your changes, and open a pull request. See CONTRIBUTING for full details. We welcome bug fixes, new LLM providers, tone improvements, and UI enhancements.
